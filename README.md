# finddd

- Mac OS用のスクリプト
- Finderウィンドウを4分割して最前面に表示する
- ホームディレクトリを少なくとも1つ表示する
- 外付けドライブがマウントされていれば優先して表示する

```
ruby finddd.rb
```
